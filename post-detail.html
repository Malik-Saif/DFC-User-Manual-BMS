Backend Facility Guides

General admin

Logging in

You can reach the Backend Login Panel by clicking here - DFC Backend. Please use the login credentials given to you by the DFC Outreach team.

Forgotten Password

If you have forgotten your password you can reset it through the ‘Forgotten Password’ link. A reset email will be sent to the email address your facility signed up with. Please check junk/spam folders if you do not receive this email.

Functionality Overview

Dashboard home - This is where you can view upcoming classes, check people in, edit and cancel classes and book on behalf of a user.

Profile - Here you can edit your facility profile, change logos, banners and important information.

Reports - This is where you will find Booking Attendance reports.

Users > User Assignment - This is only functional for DFC Fitness Villages.

Users > Instructor - Here you can input the names of the instructors that will be delivering your classes.

Users > Users - This is where you will find your facility user profile.

Schedule > Activity - This is where you input your class name & description.

Schedule > Schedule - This is where you schedule your classes to appear in the frontend booking system.

Schedule > Zones - You can create zones for your facility here, for example HIIT Studio 1

Reporting

Attendance reporting

The booking attendance report provides information about all the users that have booked your classes such as which class they have booked and on what date.

The following information can be found here

  1. Client Name
  2. Facility Name
  3. Zone Name
  4. Activity Name
  5. Activity Date
  6. Activity Time
  7. Booking Status

For a more detailed view you can download the attendance report as a .csv and other file types.

Updating your facility profile

To update your facility profile on the booking application you will need to first update it in the backend panel.

Go to ‘Users’ on the left hand panel and select ‘Users’ again.

Find your facility profile, it will be labelled with ‘Facility’ under the ‘Groups’ column. You can also use the filter on the top right of the dashboard to filter by > groups >facility.

Once you have found your facility click on the blue email address text to enter your profile dashboard.

User Tab

Business Logo: Upload your business logo here, recommended dimensions and file type are XXX

Business banner: Upload your business banner here recommended dimensions and file type are XXX

Business information: Here you can put a description of your business or facility.

Website: Enter your business’ website url here

Terms and Conditions: Here you can upload a Terms & Conditions document that will be emailed to participants with booking confirmation. Please upload in PDF Format no larger than 500kb.

Phone: Enter your business’ phone number here

Category: Here you can select what kind of business you are and what specific activity types you offer. These categories will be used in the booking application so frontend users can find the type of activities they are looking for.

All other input fields are not required and are used by HQ admin only. Please leave the following fields blank: Firstname - Lastname - Gender - DOB

Contact Tab

Please enter the contract details of the main point of contact for your facility. This will only be used by DFC Admin staff.

Address Tab

Please enter the business address for your facility, this is where user will come to participate in classes.

Gallery Tab

Here you can upload images of your facility and classes for users to see in the frontend booking application.

Please note that you are representing a government entity, and any uploaded images must adhere to strict guidelines set by DTCM. Any breach of these guidelines will.

We ask you to upload images no larger than 1mb

Managing bookings

Dashboard overview

You can click the ‘Home’ Icon to get back to this view at any time. Here you can manage all of your bookings in one place.

  1. Filter results to find a class
  2. .Edit a class time and capacity
  3. View booked users and check them in
  4. Cancel a class
  5. Book a class for a user
  6. Select an instructor for a class.

Finding a class

To find a class first navigate to the dashboard view for managing bookings, do this by clicking on the home icon in the top left of the screen.

Once here, the dashboard will show you all the bookings scheduling for the current day. You can change this to the next 7-30 days with the grey buttons at the top of the screen and even select a specific day in the calendar.

You can also search for an activity name using the ‘Choose Activity’ filter on the left hand side.

Checking in users

Once you have found your desired class, you can check in the user by clicking on the ‘User’ icon to the right of the class.

This will bring up a menu where you can search for a users name,email or phone number.

Once you have found the user you’d like to check in click the ‘Tick Icon’ on the far right of the screen and confirm your choice.

This will turn the tick green next to the users name and will signify that the user is checked in.

Cancelling a class

To cancel a class head to your facility dashboard and find the class you’d like to cancel. Click on the ‘Red X’ on the right hand side and click Ok to confirm. This cannot be undone and will send an email to all people who have booked the class to inform them the class has been cancelled.

Editing class time & capacity

To edit the the capacity or time of a class select the ‘Pencil Icon’

This will bring up a menu, change the desired details and click ‘Update’

Booking a class on behalf of a user

To book a class for a user select the blue ‘Book Class For User’ button.

You will then be prompted to input the user's email address and select the class you wish to book them onto.

Click save at the bottom of the page and the user will be booked into the activity.

Creating & Selecting an Instructor

To create an instructor profile, navigate to the menu on the left hand side, click ‘User’ > ‘Instructor’ and ‘Add new’.

Enter the instructor's name and click ‘Save’.

Navigate back to the dashboard and use the drop down menu to select a user for a specific class.

Scheduling Classes

Creating a class

To create a class first you must create an ‘Activity’ and activity ‘zone’.

To find out how to create an activity click here

To find out how to create a zone click here

Once these are created, in the left hand panel select ‘Schedule’ > ‘Schedule’ again > then ‘Add new’.

You will then be prompted to input the event details.

Facility: Select your facility from the drop down box

Zone: Select the zone your activity will take place in, eg. HIIT Studio 1

Instructions file: Instructions files are PDF’s that are attached to the users booking confirmation email. These can contain extra information about the activity such as what to bring and where to park. Instruction files are optional but recommended. Find out more about instruction files here.

Activity: Select the activity that you created for the class.

Enable Location: Enable location should only be used if your class takes place outside your designated facility. Otherwise, leave this box unchecked.

Start Date/End Date: Start Date/Time is where you input the start date of your class. If your class is a one off activity select the end date as the same date.

If your class is a recurring class you can set the end date in the future to when the last class will end.

Recurrence: For recurring classes, you can select which days of the week your class will be running. For example if your class runs every monday for 4 weeks, you can select monday in recurrence and set your end date to 4 weeks after the start date.

Duration: This is how long your class will last in minutes.

Offer online: If your class is an online class please check this box

Slots: This is how many users can attend your class.

Creating a Zone

Zones are used to direct users to the correct location within your facility. For example if you have multiple studios in your facility you can create them here.

To create a zone, in the left hand panel select ‘Schedule’ > ‘Zones’ > ‘Add new’.

Then input the ‘Zone Title’ and ‘Description’ and Click ‘Save’.

Creating an Activity

Creating an activity allows you to name your class and provide a brief description of it. You can then select this activity when scheduling a class.

To create an activity, in the left hand panel select ‘Schedule’ > Activity > ‘Add new’.

Then select your activity type, eg. yoga. Activity types are set by Dubai Fitness Challenge Admins, if there is not an activity type that suits your activity please contact your Fitness Outreach Advisor.

You can then enter the activity name and description and click ‘Save’.
